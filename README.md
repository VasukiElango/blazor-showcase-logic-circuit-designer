# Essential Studio® for Blazor - Logic Circuit Designer
The Logic Circuit Designer App is a powerful and intuitive web application built using Blazor, a modern web framework that allows developers to build interactive and dynamic web applications using C# and .NET. This application is designed to facilitate the creation, simulation, and analysis of digital logic circuits.

## Requirements to run the demo

* [System requirements](https://blazor.syncfusion.com/documentation/system-requirements/)

* [NET 8 WebAssembly Workload](https://learn.microsoft.com/en-us/aspnet/core/blazor/webassembly-build-tools-and-aot?view=aspnetcore-8.0#net-webassembly-build-tools)

## How to run the demo

Clone the repository.This repository contains Blazor Logic Circuit Designer Server app, Blazor Logic Circuit Designer WASM app.
* `Server` folder has solution and project files to run Logic Circuit Designer server app.
* `WASM` folder has solution and project files to run Logic Circuit Designer WebAssembly app.
* The Common folder contains all the common files (i.e., samples, static web assets, resources) which are applicable for Blazor Logic Circuit Designer Server demo and Blazor Logic Circuit Designer WASM demo.

### Run the demo using .NET CLI

* Open the command prompt from the demo's directory.
* Run the demo using the following command.
   
   To run .NET 8 Logic Circuit Designer Server project
   > `dotnet run --project Server/LogicCircuit.csproj`

   To run .NET 8 Logic Circuit Designer WASM project
   > `dotnet run --project WASM/LogicCircuit .sln`

### Run the demo using Visual Studio

* Open the solution file using Visual Studio.
* Press `Ctrl + F5` to run the demo.

### Run the demo using Visual Studio code

* Open the Visual Studio code from the Demo directory where the project file is present.

* Press `Ctrl + F5` to run the demo.
